## Descripción

Este proyecto corresponde al Trabajo de Desarrollo Grupal N°2 de la asignatura Métodos de Inteligencia Artificial.

El proyecto implementa una Red Neuronal utilizando TensorFlow/Keras para resolver un problema de clasificación binaria.

- TensorFlow / Keras
